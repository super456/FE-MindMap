[在线预览地址](https://super456.github.io/FE-MindMap/)

## 使用

1. 先安装 `pnpm`

```sh
npm i -g pnpm
```

2. 安装依赖

```sh
pnpm install
```

3. 生成 XMind 文件

**注意：** 请将在 `xmind` 文件夹下放置 `.xmind` 文件

```sh
pnpm gxmind
```

4. 生成 README 文件

**注意：** 将会自动生成 `README.md` 文件覆盖根目录 `README.md` 文件

```sh
pnpm greadme
```

5. 开发启动

```sh
pnpm dev
```

6. 构建

```sh
pnpm build
```

7. 预览

```sh
pnpm preview
```
